# WaypalAnalyst

A standalone React/Next.js application for hotel price analysis and comparison.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/components/Analyst/` - Analyst-specific components
- `src/api/` - API connection logic
- `src/hooks/` - Custom React hooks
- `app/` - Next.js app router pages and layouts

## Environment Variables

Create a `.env.local` file with:
```
NEXT_PUBLIC_AGENT_BACKEND_URL=your_backend_url
```
